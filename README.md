# KargoGuard

KargoGuard is an AI-powered cargo protection and tracking platform.
It uses advanced image processing and AI to analyze cargo condition, Web3 technologies for transparent tracking, and a comprehensive API for system integration.

## Architecture

- **KargoGuard.API**: Core backend services built with .NET.
- **KargoGuard.AI**: AI microservices for image analysis built with Python.
- **KargoGuard.Web3**: Smart contracts for transparent tracking.
- **KargoGuard-Mobile / MobileApp**: Mobile application for end-users and couriers.
- **vite-project (Dashboard)**: Web dashboard for administrative and operational monitoring.
